# 🎫 Incident Ticket Simulator (Python)

A Python-based simulation of an IT Service Management (ITSM) system that mimics incident lifecycle used in NOC/ServiceNow environments.

---

## Features

- Create incident tickets
- Assign priority (P1–P4)
- Update ticket status (Open, In Progress, Closed)
- View all tickets
- Simulates real ITIL workflow

---

## Tech Stack

- Python
- File Handling

---

## How to Run

1. Add sample tickets in `tickets.txt`
2. Run:
   python simulator.py

---

## Use Case

Simulates real-world incident management system used in:

- NOC (Network Operations Center)
- IT Support teams
- ServiceNow environments
  
## Features Demonstration

- Create Ticket → Assign priority (P1–P4)
- Update Ticket → Change status (Open → In Progress → Closed)
- View Tickets → Track all incidents
